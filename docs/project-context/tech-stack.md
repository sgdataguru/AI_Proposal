# Preferred Technology Stack

## Cloud Platform

### Primary Cloud Provider: **Microsoft Azure**

**Rationale**:
- Existing Microsoft 365 and Dynamics CRM infrastructure
- Enterprise Agreement in place with favorable pricing
- Strong integration with existing authentication (Azure AD)
- GDPR compliance for European operations
- Local data residency requirements for multiple markets

**Azure Subscription Requirements**:
- Production and non-production environments
- Multi-region deployment capability (US, EU, APAC)
- Cost management and monitoring tools

**Preference for Microsoft Fabric**
- Modern architecture
- Built for medallion architecture
